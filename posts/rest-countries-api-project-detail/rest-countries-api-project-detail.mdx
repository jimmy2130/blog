### 墨西哥捷運簡介

墨西哥城捷運，為墨西哥主要的公共運輸系統，其路線長度共有226.5公里，在總計12條路線中，有195座車站。所有路線皆以不同的顏色和編號來區分，其中，最新一條捷運為12號線，以金色表示。

12號線於2012年10月通車，行駛自米斯科阿克(Mixcoac)至特拉胡阿科(Tláhuac)，全長約24公里，共有20座車站，包含9座地下車站、9座高架車站和2座平面車站。儘管12號線是最新的捷運，該線曾經歷多次的整修。舉例來說，2014年至2015年，12號線因為結構性故障而暫時關閉，2017年則是因為規模7.1的大地震而造成四個車站關閉。2017年時任的捷運局長加維諾(JorgeGaviño)曾經指出:「12號線天生有著一輩子也無法解決的地方性問題，並且需要永久地進行維護。」
